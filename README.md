## 目前的功能

### 待办事项
#### 通知和提醒
- 通过磁贴显示未完成的待办事项（Win10）
- 通过通知在事件到期前提醒
- 每日定时发送通知呈上目前待办事项的情况

#### 事项显示
- 进度条显示时间进度（真实deadline🤣）
- 事项优先级显示
